# 🐾 Rently – Service Animal Platform

**Rently** is a web-based service animal rental and support platform designed to connect users with licensed service animals and handlers. Built with [Vite](https://vitejs.dev/), [React](https://reactjs.org/), and TypeScript, the platform aims to provide accessibility, transparency, and ease of use for individuals in need of temporary service animal support.

## 🚀 Live Website

👉 Visit the live site:  
**[https://muroyyy.github.io/rently/](https://muroyyy.github.io/rently/)**

## 🛠️ Tech Stack

- ⚡️ Vite
- ⚛️ React
- ⌨️ TypeScript
- 💨 Tailwind CSS
- 🧪 ESLint

## 📦 Scripts

```bash
npm run dev       # Start local development server
npm run build     # Build for production
npm run preview   # Preview production build
npm run deploy    # Deploy to GitHub Pages
```

## 🗂️ Deployment

This project uses GitHub Actions to automatically build and deploy the site to the gh-pages branch.
Deployment is triggered on every push to the main branch.

## Folder Structure

```bash
src/            # Source code (components, pages)
public/         # Static files
dist/           # Built assets for production (auto-generated)
```
